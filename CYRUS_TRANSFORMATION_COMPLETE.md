## 🚀 How to Use

### 1. Setup Local AI (Recommended First Step)
```bash
chmod +x scripts/setup-local-llm.sh
./scripts/setup-local-llm.sh
```

This installs:
- Ollama service
- Llama 3.2, 3.1, CodeLlama, LLaVA models
- Custom Cyrus models (General, Code, Medical, Legal)
- Environment configuration

### 2. Configure Environment

**Option A: Local Only (Zero Cost)**
```bash
echo "USE_LOCAL_LLM=true" >> .env
```

**Option B: Maximum Intelligence**
```bash
cat >> .env <<EOF
OPENAI_API_KEY=sk-...
ANTHROPIC_API_KEY=sk-ant-...
GOOGLE_AI_API_KEY=...
USE_LOCAL_LLM=true
CYRUS_MULTI_MODEL_STRATEGY=specialized
EOF
```

### 3. Start Cyrus
```bash
npm install
npm run dev
# Open http://localhost:3020
```

### 4. Verify Everything Works
```bash
# Check system status
curl http://localhost:3020/api/cyrus/status

# Expected: All systems operational
{
  "status": "operational",
  "systems": {
    "local-llm": true,
    "multi-model": true,
    "voice-stt": true,
    "voice-tts": true,
    "learning": true,
    "evolution": true
  }
}
```
